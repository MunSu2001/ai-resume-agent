# AI 기반 자기소개서 개선 에이전트 시스템

## 📖 프로젝트 소개

사용자가 작성한 자기소개서를 입력하면  
기업 정보와 최신 뉴스 데이터를 반영하여  
AI가 자기소개서를 자동으로 개선해주는 시스템입니다.

단순 문장 수정이 아니라,
- 기업 분석
- 최신 이슈 반영
- 직무 적합성 개선

**멀티 에이전트 기반 자기소개서 생성 시스템**입니다.

## ❗ 문제 정의

- 지원자는 기업 정보를 충분히 반영하지 못한 자기소개서를 작성하는 경우가 많음
- 최신 뉴스 및 기업 방향성이 반영되지 않음
- 자기소개서 첨삭은 주관적이고 일관성이 부족함

👉 이를 해결하기 위해 AI 기반 자동 첨삭 시스템을 설계

## 💡 해결 방법

멀티 에이전트 구조를 활용하여 문제를 해결했습니다.

- 기업 정보 에이전트 → 기업 데이터 수집
- 뉴스 에이전트 → 최신 뉴스 및 전망 분석
- 자기소개서 에이전트 → 문장 개선 및 평가
- 최종 에이전트 → 모든 정보를 종합하여 최종 자기소개서 생성

## 🏗️ 시스템 구조

```
User Input
│
▼
[ Parallel Agent ]
├── company_agent (기업 정보 수집)
├── news_agent (뉴스 분석)
└── resume_agent (자기소개서 수정)
│
▼
[ Final Agent ]
│
▼
Final Resume Output
```


## 🚀 주요 기능

- 자기소개서 자동 첨삭
- 기업 정보 자동 수집 (종목코드 기반)
- 최신 뉴스 검색 및 요약
- 멀티 에이전트 기반 병렬 처리
- 최종 자기소개서 자동 생성

## 🛠️ 기술 스택

- Python
- Google ADK (Agent Development Kit)
- Gemini API
- MCP (FastMCP)
- Pandas
- HTTPX

## ⚙️ 실행 방법

### 1. 환경 변수 설정 (.env)
API KEY 설정 필요
GOOGLE_API_KEY=your_api_key

### 2. 패키지 설치
pip install -r requirements.txt

### 3. 서버 실행
python main.py

## 🧪 사용 예시

입력:
"삼성전자 AI 직무 지원 자기소개서 ..."

출력:
- 기업 정보 반영
- 최신 뉴스 반영
- 문장 개선된 자기소개서
